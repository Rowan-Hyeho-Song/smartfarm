실증 실험 계획서 — CO₂ 제어 A/B

데이터로 강하게 시사된 가설(CO₂ 유지가 생장 손실을 막는다)을 무작위 배정 실험으로 확정하기 위한 실행 계획입니다. 온실이 확보되면 이 프로토콜대로 바로 착수할 수 있게 정리했습니다.

계획 요약

  • 목적 — 관측·시뮬로 얻은 "CO₂ ↑ → 생장 ↑" 가설의 인과를 무작위 실험으로 확정.
  • 설계 — 온실을 처치(CO₂ 700~900ppm) / 대조(자연 ≈450ppm)로 무작위 배정, 6~8주 실증.
  • 측정 — 현 데이터와 동일 항목(초장·엽병장·착과수 등) 주 1회 조사.
  • 분석 — 발달단계 통제 후 처치−대조 차이 검정. 예측 모델로 사전 표본수(power) 산정.

Hypothesis

가설 — 무엇을 검정하나

주가설 (H1)

주간 CO₂를 700~900ppm으로 유지하면, 자연 상태(≈450ppm) 대비 초장·엽병장 생장이 증가한다. (관측 준-A/B +1.57cm, 반사실 dose-response와 일치)

부가설 (H2) — 포화 여부

효과는 포화형이라, 900ppm 이상으로 더 올려도 추가 이득은 작다. (비용 대비 효과의 최적점 탐색)

Protocol

설계 — 4단계 프로토콜

  1. 온실 무작위 배정

    동일 품종·정식일·온실규격을 갖춘 온실들을 처치군/대조군에 무작위 분배. 반복(온실 수)을 최대한 확보(권장 각 군 ≥3동). 온실 간 위치 편향을 막기 위해 블록 배정.

  2. 처치 정의 (한 가지만 다르게)

    처치군: 주간 CO₂ 700~900ppm 시비. 대조군: 자연 상태. 온도·관수(EC/PH)·광 관리는 양 군 동일하게 유지 → CO₂ 효과만 격리.

  3. 기간·측정

    6~8주 실증. 현 데이터와 동일 항목·동일 주기(주 1회: 초장·엽병장·엽수·관부직경·착과수). 환경은 시간별 자동 로깅(온·습도·CO₂·일사·급액).

  4. 분석

    발달단계(조사회차) 통제 후 처치−대조 차이를 t검정/혼합효과모형으로 검정. 실험 전 예측 모델로 표본수(power) 산정해 몇 동·몇 주가 필요한지 사전 계산.

Metrics

측정 지표 & 성공 기준

구분지표성공 기준(예시)
1차 결과초장 · 엽병장처치군이 대조군보다 유의하게 큼 (p<0.05)
2차 결과착과수 · 수확량차이 관찰(발달단계 지배라 부수적)
비용 효율ppm당 생장 이득포화점(추가 이득 < 비용) 식별
모델 검증예측 vs 실측berry_ai 예측 구간 안에 실측이 ~80% 포함
현 데이터와 동일한 항목·주기로 측정하면, 실증 데이터를 data_prep.py에 바로 넣어 모델을 재학습·검증할 수 있습니다. 실증이 곧 모델 업데이트로 이어지는 구조.

Risk

리스크 & 대비

온실 간 고유 편차 — 무작위 배정 + 반복(여러 동)으로 상쇄. 사전에 온실별 baseline 측정.
처치 누출(대조에 CO₂ 유입) — 물리적 격리·모니터링으로 방지.
표본 부족 — 온실 수가 적으면 검정력 부족 → 실험 전 power 계산으로 기간·반복 조정.
계절 교란 — 처치·대조를 동시 기간에 운영해 계절 효과를 양 군에 공통으로.
