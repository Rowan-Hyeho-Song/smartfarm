베리디시전 · 룰 베이스 제어

데이터 상관관계로 지정한
룰 베이스 제어 규칙

보유 데이터의 환경↔생육 상관과 환경 지표 간 상관을 근거로, "이런 조건이면 이렇게 제어한다"는 규칙을 정리했습니다. 단, 계절 교란을 걷어낸 '진짜 신호'만 규칙으로 채택했습니다.

핵심 4가지

  • 가장 강한 레버 = CO₂ — 발달단계를 제거한 뒤에도 CO₂ ↑ → 잎·줄기 생장 ↑(초장 +0.40, 엽병장 +0.39). 룰 1순위.
  • 복합(다변량)이 진실 — 다른 변수를 통제한 부분상관을 보면 CO₂만 독립 레버로 살아남고(+0.26), VPD·습도는 CO₂·온도에 딸린 그림자(부분상관 ≈0). 짝 상관만 믿으면 오판.
  • 제어는 서로 얽힌다 — 난방→온도↑→습도↓·VPD↑, 환기→온도↓·CO₂↓. 한 밸브만 돌리면 다른 게 딸려옴.
  • 정직하게 — 착과수·관부직경은 단기 환경제어로 조절이 약함(발달단계 지배) → 환경 룰보다 적과·재배관리로.

Caution

먼저 — 원시 상관은 함정이다

원시 데이터에서 "내부온도 ↑ → 착과수 ↓"(−0.52)처럼 보입니다. 하지만 이건 온도가 착과를 줄여서가 아닙니다. 겨울작이라 시간이 갈수록 (a) 식물이 자라고 (b) 날이 추워지기 때문에 생기는 가짜 상관(계절 교란)입니다. 이대로 "온도를 낮추면 착과가 는다"는 룰을 만들면 농업적으로 틀린 제어가 됩니다.
그래서 발달단계(조사회차)를 제거하고 — 즉 '같은 재배 주차에서 온실 간 차이'만 봐서 — 순수한 환경 효과만 남긴 상관을 규칙의 근거로 씁니다.

Signal

① 발달단계를 걷어낸 '진짜' 환경 → 생장 상관

초장 기준 (엽병장도 거의 동일). 오른쪽(+)=높을수록 생장↑, 왼쪽(−)=생장↓.

발달단계(조사회차) 제거 후 Spearman 상관 · 초장 기준. CO₂가 가장 강한 양(+) 신호.

환경 요인초장엽병장착과수관부직경해석
내부 CO₂+0.40+0.39약함약함광합성 원료 → 잎·줄기 생장 촉진
내부 온도−0.32−0.33약함약함같은 단계선 저온 쪽이 신장 큼(도장 관점)
VPD(건조)−0.22−0.23약함약함너무 건조하면 증산 억제 → 생장 저해
급액 PH——−0.23—PH 벗어나면 흡수 저하
착과수·관부직경은 발달단계를 걷어내면 환경 상관이 거의 사라집니다(|r|<0.2). 즉 단기 환경제어로 착과수를 직접 늘리기는 어렵고, 발달단계·적과·양분으로 관리하는 게 맞습니다. 룰로 과장하지 않습니다.

Multivariate

② 복합(다변량) 상관 — 짝 상관을 넘어

"CO₂만 올리면 되나?" 아닙니다. 변수들이 서로 얽혀 있어, 다른 변수를 통제한 부분상관과 상호작용을 봐야 진짜 레버가 보입니다.

내부 CO₂+0.40+0.26온도·일사통제해도 살아남음 → 독립 레버
내부 온도−0.32−0.20CO₂·습도약해지지만 유효
VPD−0.16−0.08CO₂거의 사라짐 → CO₂의 그림자
내부 습도+0.13+0.06온도사라짐 → 온도의 그림자
해석 — VPD·습도는 '독립적으로' 생장을 움직이는 게 아니라, CO₂·온도가 바뀌면 딸려 움직이는 값이었습니다. 짝 상관(−0.22)만 보고 "VPD를 생장 레버로" 쓰면 헛다리 — 생장 목적엔 CO₂를 직접 제어하고, VPD는 병해·안전용으로 둡니다.

Coupling

③ 제어 레버는 서로 얽혀 있다

시간별 환경 지표 간 상관(온실1~4). 한 밸브를 돌리면 다른 값이 딸려옵니다.

난방 ↑내부온도 ↑습도 ↓ (온도-습도 −0.48) · VPD ↑ (온도-VPD +0.59) → 공기가 건조해짐
환기 ↑내부온도 ↓CO₂ ↓ (온도-CO₂ −0.52, 외기 유입) · 습도 ↑

관수(EC/PH)양분·산도기후와 거의 독립(상관 ≈ 0) → 온·습도와 따로 조절 가능
핵심 상충: CO₂ 시비 ↔ 환기가 반대로 움직입니다. 그래서 "낮이지만 환기 최소인 시간대(광 있고 온도 안정)"에 밀폐+시비하는 게 최적입니다.

Honesty

한계와 다음 단계

상관은 인과가 아닙니다 — 특히 온도 음(−) 신호는 계절 교란을 완전히 배제하지 못했을 수 있어, '주의' 등급으로 뒀습니다.
데이터가 4온실 · 겨울 15주뿐이라 규칙의 임계값(예: 700~900ppm)은 농업 표준범위를 참고로 결합한 값입니다.
다음 단계: A/B 제어 실험(CO₂ 시비 온실 vs 대조)으로 룰의 인과 효과를 실측 검증.
