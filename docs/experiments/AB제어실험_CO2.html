A/B 제어 실험 — 제어 레버 검증

룰 베이스 제어의 핵심 레버(CO₂ · 온도)가 정말 생장을 움직이는지, 살아있는 온실 없이 데이터로 A/B를 재현해 검증했습니다. 관측 기반 준-A/B와 모델 기반 반사실 시뮬레이션, 두 방법으로 교차 확인합니다.

핵심 3가지

  • 방향은 확실 — 두 방법 모두 CO₂ ↑ → 생장 ↑. 관측 준-A/B에서 고CO₂ 그룹이 초장 +1.57cm 더 큼 (p<0.001).
  • 비대칭(포화) 반응 — CO₂를 떨어뜨리면 손실이 큼(−100ppm → −1.4cm), 올리면 이득은 점점 작아짐(+200ppm → +0.15cm).
  • 제어 함의 — "무리한 고농도 시비"보다 적정선(주간 700~900ppm) 유지·하락 방지가 가성비 높음.

Method

방법 — 왜 두 가지로 재현하나

진짜 A/B 제어 실험은 살아있는 온실을 무작위로 처치/대조로 나눠야 합니다. 우리는 과거 관측 데이터만 있으므로, 실제 트라이얼 대신 다음 두 가지 대리(proxy) 실험으로 검증합니다. 인과의 완전한 증명이 아니라 강한 가설 검증입니다.
① 관측 기반 준-A/B

같은 발달단계에서 'CO₂ 높은 온실-주간(A)' vs '낮은(B)'을 갈라 실제 생장을 비교. (자연 실험)

② 모델 반사실 시뮬레이션

학습 모델로 "CO₂를 +Δ 시비했다면?"을 가상으로 예측. 다른 조건은 고정(순효과 격리).

Result A

① 관측 기반 준-A/B — 고CO₂ vs 저CO₂

발달단계를 통제한 뒤, CO₂ 상위 그룹(A) − 하위 그룹(B)의 생장 차이. 값은 단계평균 대비 편차(cm).

초장 (단계평균 대비 편차). A−B = +1.57cm, p<0.001

지표A 고CO₂B 저CO₂차이p
초장+0.58−1.00+1.57<0.001
엽병장+0.57−0.98+1.54<0.001

같은 재배 주차에서 CO₂가 높았던 온실-주간이 초장·엽병장 모두 약 1.5cm 더 컸고, 통계적으로 매우 유의(p<0.001)합니다. 표본 A=380 / B=220.

Result B

② 모델 반사실 — CO₂ 시비량별 예측 생장 (dose-response)

다른 조건은 고정하고 CO₂만 ±Δ 조작했을 때, 모델이 예측한 초장 변화.

초장 변화 예측 (cm) · 0ppm(현 상태) 기준. 왼쪽 급락 / 오른쪽 완만 = CO₂ 포화 반응.

③ 온도 레버 — '양날의' 레버 (조건부)

CO₂와 달리, 온도는 방향은 뚜렷하지만 '높일수록 좋은' 게 아닙니다.

온도 A/B · 초장고온 A저온 B차이p
전체−0.48+0.59+1.07<0.001
저습(건조)−0.44+0.52+0.96<0.001
고습(습함)−0.52+0.64+1.16<0.001

CO₂의 포화형과 달리 대칭·선형. 온도를 내리면 신장이 늘고, 올리면 준다.

초장↑이 항상 '좋음'은 아님 — 과도한 신장은 도장(연약 생장)입니다. 즉 온도는 CO₂처럼 "높일수록 이득"인 레버가 아니라, 식물 체형(신장 vs 충실)을 조절하는 트레이드오프 레버입니다. 저온장해 하한을 지키는 선에서 '충실한 생육'을 목표로 미세조정해야 합니다.

Synthesis

종합 — 두 결과가 말하는 것

CO₂ — 두 방법 모두 CO₂↑ → 생장↑로 일치. 다만 포화형이라, 가장 실행 가능한 메시지는 "고농도 시비"보다 CO₂가 떨어지지 않게 유지(주간 700~900ppm)하는 것.
온도 — 저온일수록 신장↑(대칭·유의)이지만 초장↑이 곧 좋음은 아님(도장). 온도는 '높일수록 이득'이 아니라 체형을 조절하는 트레이드오프 레버.
→ 두 레버의 성격이 다르다: CO₂는 "유지/확보", 온도는 "균형 조절".

Honesty & Next

한계와 진짜 A/B 프로토콜

관측 준-A/B는 잔여 교란(고CO₂ 온실이 다른 관리도 잘했을 수)이 남고, 모델 시뮬은 상관 기반·외삽이라 인과를 완전히 증명하지 못합니다. 진짜 인과는 무작위 배정 실험으로만 확정됩니다.
실제 A/B 제어 프로토콜 (다음 실증 단계)
  1. 온실 무작위 배정 — 동일 품종·동일 정식일 온실을 처치(CO₂ 시비)/대조로 랜덤 분배. 온실 수(반복) 최대한 확보.
  2. 처치 정의 — 처치군: 주간 700~900ppm 유지, 대조군: 자연 상태(≈450ppm). 나머지 온도·관수는 동일.
  3. 기간·측정 — 6~8주간 주 1회 초장·엽병장·착과수 조사(현 데이터와 동일 항목).
  4. 분석 — 발달단계 통제 후 처치−대조 차이 검정. 예측 모델로 사전 표본수(power) 계산.
